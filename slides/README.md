# Slides

Drop your presentation files in this folder.

Suggested layout:

```
slides/
├── neurodesk-ms-workshop.pptx     # source slide deck
├── neurodesk-ms-workshop.pdf      # exported PDF (recommended for sharing)
└── images/                        # screenshots, figures used in slides
```

## Suggested slide outline

1. **Why we're here** — clinical motivation in MS
2. **The reproducibility problem** — why neuroimaging pipelines break
3. **What is Neurodesk?** — architecture, modules, three deployment options
4. **Live demo: Neurodesk Play in 60 seconds**
5. **MS-relevant tools available** — FSL, FreeSurfer SAMSEG, SCT, qMRLab
6. **Workflow walkthrough** — preprocessing → lesion seg → atrophy → cord → qMRI
7. **Clinical implications** — biomarkers, trials, multi-centre studies
8. **Limitations & caveats** — clinico-radiological paradox, scanner harmonisation
9. **Resources & next steps** — Neurodesk docs, community, contributions

## Tips

- Export PDF before the workshop in case the PPTX font/animations break on the venue laptop.
- Embed YouTube/animations as still images with a "click to play" link rather than relying on network.
- Include a slide with a QR code linking to this repo.
